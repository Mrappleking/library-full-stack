## 封面图片

封面图片存储在 `src/main/resources/static/covers/` 目录下：

```
covers/
├── 1-算法导论.jpg
├── 2-深入理解计算机系统.jpg
├── 3-机器学习.jpg
├── ...
└── {UUID}-{title}.{ext}
```

前端通过 `<img :src="book.cover">` 渲染封面图片。

## 数据重置

如需重置数据库，执行以下步骤：

```bash
# 删除数据库
DROP DATABASE IF EXISTS library;

# 重新创建
CREATE DATABASE library DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# 导入schema和数据
mysql -h127.0.0.1 -uroot -p library < schema.sql
mysql -h127.0.0.1 -uroot -p library < seed.sql
```
